## 3. Pilot MoUs with Small Units/Offices

### 3.1 Target Profile

Ideal pilot partners are small government units that:
- Handle significant document volumes (reports, manuals, case files, correspondence)
- Have 5–50 users (manageable pilot scope)
- Lack enterprise AI tools (greenfield opportunity)
- Have authority to sign MoUs without lengthy procurement (micro-purchase threshold or innovation authority)
- Are willing to provide testimonials and case studies

### 3.2 Target Categories

| Category | Examples | Document Types |
|----------|---------|---------------|
| **Inspector General offices** | Agency OIGs, state auditor offices | Audit reports, investigation files, compliance docs |
| **JAG / Legal offices** | Military legal, state AG offices | Case files, legal research, regulations |
| **Small agency HQs** | USAGM, CNCS, MCC, USIP | Policy documents, reports, correspondence |
| **Congressional offices** | Committee staff, CRS | Research reports, hearing transcripts, legislation |
| **State legislative staff** | Bill analysis offices, fiscal offices | Bills, fiscal analyses, committee reports |
| **Public library systems** | Special collections, archives | Digitized collections, reference materials |
| **University research offices** | Sponsored research, grants admin | Grant proposals, compliance docs, research papers |

### 3.3 MoU Structure

| Element | Detail |
|---------|--------|
| **Duration** | 90 days (with option to extend) |
| **Cost** | Free pilot (or micro-purchase threshold: $10K federal, varies by state) |
| **Scope** | Single workspace, up to 25 users, up to 1,000 documents |
| **Deliverables** | Configured workspace, user training (1 hr virtual), weekly check-in |
| **Success Metrics** | User adoption rate, query volume, user satisfaction score, time-saved estimate |
| **IP/Data** | Customer retains all data; data deleted at pilot end unless contract follows |
| **Exit** | Customer can request data export and deletion at any time |

### 3.4 Pilot-to-Contract Conversion

1. **Week 1–2**: Onboarding — workspace setup, document upload, user training
2. **Week 3–8**: Active use — weekly check-ins, usage analytics, feedback collection
3. **Week 9–10**: Assessment — user survey, time-saved analysis, ROI calculation
4. **Week 11–12**: Decision — present results to contracting authority; proposal for paid contract
5. **Post-pilot**: Case study (with permission), testimonial, reference for future bids
