## Phase 2: Personnel (Complete Before CJI Access)

- [ ] CEO signs CJIS Security Addendum (upon first customer engagement)
- [ ] CEO completes fingerprint-based background check (state-specific)
- [ ] CEO completes CJIS security awareness training
- [ ] All engineers with production access sign addendum
- [ ] All engineers complete fingerprint-based background checks
- [ ] All engineers complete CJIS security awareness training
- [ ] Personnel tracker fully populated

## Phase 3: Customer Engagement (Per Customer)

- [ ] Identify target customer agency
- [ ] Initial meeting / demo with agency decision-maker
- [ ] Agency introduces Latent Archon to state CSA
- [ ] MCA customized and executed with agency
- [ ] State CSA provides fingerprint submission instructions
- [ ] All personnel submit fingerprints to state CSA
- [ ] Background check results received and cleared
- [ ] CJI access provisioned (production workspace for customer)
- [ ] Customer agency onboarded (users, documents, workspaces)

## Phase 4: Audit Readiness (Ongoing)

- [ ] All signed addenda on file (encrypted storage)
- [ ] All background check confirmations on file
- [ ] All training records current in Drata
- [ ] Personnel tracker up to date
- [ ] Audit log retention verified (1+ year)
- [ ] Incident response contacts include CJIS-specific entities
- [ ] First state CSA audit completed (triennial cycle begins)
